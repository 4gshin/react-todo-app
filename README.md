# React Todo App

A clean and responsive Todo App built with React featuring task editing, filtering, dark mode and localStorage support.

## ✨ Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Active / Completed)
- Clear completed tasks
- Dark / Light theme
- LocalStorage persistence

## 🛠 Tech Stack

- React
- JavaScript (ES6+)
- CSS
- LocalStorage API

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/4gshin/react-todo-app.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```
